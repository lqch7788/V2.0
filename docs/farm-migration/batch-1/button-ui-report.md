## 五、修复优先级

### P0（必修复，核心交互）
- 排班主页面缺失 15 个按钮（取消批量、详情侧边栏操作、3 个 Modal 确认/取消按钮）
- 排班 3 个 Modal 完全缺失（批量编辑 4 按钮、删除警告 2 按钮、导出格式 2 按钮）

### P1（重要交互）
- ScheduleCalendar 子组件 11 个按钮（月/周/日切换、上一天/今天/下一天）
- ScheduleTable 子组件 14 个按钮（搜索、筛选、分页）
- ShiftEditor 4 个按钮（编辑模式、保存、取消、颜色选择 8 个）

### P2（次要交互）
- 表格列内操作（行点击、复选框）
- 班次图例
- 排班详情侧边栏

## 总结

**班组分配按键 100% 对齐**（P0 已修复完毕）
**排班调度按键 18% 对齐**（66 个中缺失 54 个）

排班调度是**最大缺口**，主要原因：
1. V2.0 把 ScheduleTable / ScheduleCalendar 集成在 Schedule.vue 内部，未拆分子组件
2. V2.0 缺失 3 个 Modal（批量编辑、删除警告、导出格式）
3. V2.0 缺失侧边栏 3 块（排班详情、调班申请、班次图例）
4. V2.0 缺失日历/表格的工具栏（搜索、筛选、分页、视图切换）
